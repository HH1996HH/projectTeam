将项目通过git进行管理
1.自己创建好基本项目结构
2.将我们的项目变成一个git仓库->>git init
3.因为我们代码仓库只需要存放我们的代码 像class文件等编译器自身产生的文件都不需要放入仓库中
而且如果放入还会经常出现代码冲突,解决方案就是在忽略文件中进行设置
.gitignore
4.在设置好忽略文件后我们就可以将我们的代码提交到我们的本地仓库了
这时候对于我们的文件我们需要先通过
git add 要添加的文件的路径 来将我们要放入本地仓库的代码进行添加操作,接下来通过git commit -m "提交信息"来真的将我们的项目代码提交到本地仓库,
这时候才是真的将代码保存在本地仓库了,我们如果希望跟别人共同进行代码的管理急需要将我们
本地仓库和远程git仓库进行连接并将代码推送到远程服务器上 这样才能跟别人进行共同开发
git remote add origin 远程服务器上仓库的地址
这样连接好了之后 我们就可以进行项目从本地到服务器的推送了
git push -u origin master


合作开发的步骤
1.框架搭建好 上传至github
2.邀请组员成为协作者
3.组员git clone 项目克隆
4.coding完成 进行上传
5.先pull拉取 将服务器上最新的代码拉取到本地
6.push推送自己的代码到服务器
